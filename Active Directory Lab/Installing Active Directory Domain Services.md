# Step 1: Connect to server
- Use Remote Desktop (RDP) to connect to server
- Log in with local administrator account

# Step 2: Open Server Manager
- Server Manager should open automatically after login
- If not, open from start menu

# Step 3: Add Roles and Features
- Select Add Roles and Features
- On 'Before you begin' screen, click next
- Select Role-based or Feature-based installation
- Select local server (default)

# Step 4: Server Roles
- Check Active Directory Domain Services
- When prompted, select Add Features

# Step 5: Features
- Leave all features at their default selections

# Step 6: AD DS Information
- Review the information page
- Select Next

# Step 7: Confirmation:
- Confirm the role selection
- Select Install
